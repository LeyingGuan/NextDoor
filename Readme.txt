
# Next-door Analysis

This repositoty contains source packages for R package nextdoor.

## Installation:
library(devtools)

install_github("LeyingGuan/NextDoor/nextdoor")

library(nextdoor)

or download the zip file nextdoor_0.1.0.tar.gz and install it in R locally.
## Overview

There two main functions in nextdoor:

nextdoor.glmnet

nextdoor

Details of how to use the functions can be found in the help page coming with each of the function in R, demonstrated with detailed examples.
